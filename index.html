Soroban Sentry is a unified infrastructure protocol that provides institutional-grade compliance, security monitoring, and immutable audit trails for smart contracts on Stellar.

How It Works Explore Lens CLI

The Institutional Bridge

To achieve mass adoption of tokenized assets, Stellar requires a compliance layer that is as decentralized as the network itself.

The Challenge

Current on-chain compliance is often hard-coded into asset contracts, making it rigid, difficult to audit, and expensive to update as regulations shift.

The Sentry Solution

Sentry decouples compliance logic from the asset layer. By using a modular "Guard" architecture, issuers can update regulatory rules in real-time without redeploying their core tokens.

Modular Guard Architecture

A multi-layered defense and compliance system designed for the Soroban VM.

1

The Sentry Hub

A centralized on-chain registry where assets register their compliance requirements. Every transaction triggers a "Sentry Hook" before finalization.

2

Guard Modules

Isolatable logic containers (WASM) that perform specific checks: KYC status, Geofencing, Velocity limits, or AML pattern matching.

3

Audit Engine

Shared Rust types (XDR) ensure that every check is mirrored in a persistent off-chain indexer, creating a regulator-friendly immutable trail.

What Sentry Can Do

Comprehensive features tailored for Institutional DeFi and Asset Issuers.

🌍

Global Geofencing

Automatically restrict asset transfers based on the jurisdictional eligibility of the sender and receiver, ensuring compliance with local laws.

📈

Velocity & Volume Limits

Prevent flash-crashes and suspicious activity by enforcing daily, weekly, or per-transaction value limits across your entire asset pool.

🆔

KYC/AML Integration

Plug into existing identity providers to ensure only verified participants can interact with permissioned liquidity pools.

🚨

Circuit Breakers

Sentry monitors for anomaly patterns. If a security breach is detected, it can automatically pause contract execution to protect user funds.

📜

Regulatory Reporting

Generate one-click audit reports via the Sentry Indexer, providing full transparency to regulators without exposing sensitive user data.

🔗

XDR Parity

Built with shared Rust common types, ensuring 100% data integrity between the Soroban contract, the indexer, and the auditor's CLI.
